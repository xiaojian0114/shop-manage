# 电商系统后台管理平台

基于 Vue 3 + TypeScript + Element Plus 的后台管理系统前端项目。

## 功能模块

- ✅ 用户认证（登录、获取用户信息）
- ✅ 用户管理（列表、详情、编辑、删除、启用/禁用）
- ✅ 店铺管理（列表、详情、编辑、删除、审核通过/驳回）
- ✅ 商品管理（列表、详情、编辑、删除、上架/下架）
- ✅ 订单管理（列表、详情、状态更新、删除）
- ✅ 数据统计（用户、店铺、商品、订单、销售统计）

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **TypeScript** - 类型安全的 JavaScript
- **Vite** - 下一代前端构建工具
- **Vue Router** - 官方路由管理器
- **Pinia** - 状态管理库
- **Element Plus** - 基于 Vue 3 的组件库
- **Axios** - HTTP 客户端

## 环境配置

### API 基础地址

默认后端地址：`http://localhost:8080`

如需修改，请编辑 `src/api/request.ts` 中的 `baseURL`。

### 登录账号

根据 API 文档，需要使用 `role='admin'` 的用户账号登录。

默认测试账号：
- 手机号：`13800138000`
- 密码：`123456`

## 功能说明

### 1. 登录认证

- 支持手机号和密码登录
- 登录成功后自动保存 Token
- Token 自动添加到请求头中
- Token 过期自动跳转登录页

### 2. 用户管理

- 用户列表分页展示
- 支持按手机号/昵称搜索
- 支持按角色筛选（user/merchant/admin）
- 编辑用户信息（昵称、头像、角色、状态）
- 启用/禁用用户
- 删除用户

### 3. 店铺管理

- 店铺列表分页展示
- 支持按店铺名称搜索
- 支持按状态筛选（审核中/已通过/已驳回）
- 编辑店铺信息
- 审核通过/驳回店铺
- 删除店铺

### 4. 商品管理

- 商品列表分页展示
- 支持按商品名称搜索
- 支持按店铺ID筛选
- 支持按上架状态筛选
- 编辑商品信息（名称、图片、价格、库存、上架状态）
- 上架/下架商品
- 删除商品

### 5. 订单管理

- 订单列表分页展示
- 支持按订单号搜索
- 支持按订单状态筛选
- 支持按店铺ID和用户ID筛选
- 查看订单详情（包含订单项）
- 更新订单状态
- 删除订单

### 6. 数据统计

- 用户统计（总数、各角色数量）
- 店铺统计（总数、各状态数量）
- 商品统计（总数、上架/下架数量）
- 订单统计（总数、各状态数量）
- 销售统计（总销售额、已完成订单数）

## 注意事项

1. **后端服务**：确保后端服务已启动并运行在 `http://localhost:8080`

2. **跨域问题**：如果遇到跨域问题，需要在后端配置 CORS

3. **Token 存储**：Token 存储在 localStorage 中，刷新页面不会丢失

4. **路由守卫**：未登录用户访问需要认证的页面会自动跳转到登录页

5. **错误处理**：所有 API 请求都有统一的错误处理和提示

## 开发规范

- 使用 TypeScript 进行类型约束
- 使用 Composition API 编写组件
- API 请求统一封装在 `api` 目录
- 状态管理使用 Pinia
- UI 组件使用 Element Plus
